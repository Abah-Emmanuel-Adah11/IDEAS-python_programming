Problem Definition

  • This project seeks to predict the best location for purchasing housing property in Nigeria, based on budget and order features using machine learning.

  • The project at hand is a classification problem. Since it aims at predicting a categorical outcome (town), which is the Town based on numerical and categorical features such as price, title, bathrooms, bedrooms, and parking space.

Data

Data for this project will be obtained from the online platform 'Kaggle.com'

Data Dictionary (Features and Target Variable)

Dataset Includes:

The following features will be used to predict our target variable (Predicted Town) in our project:

Title (string) - The house type e.g Semi-Detached Duplex, Detached Duplex, Bongalow ..

Parking Space (int) - The number of parking spaces within the property e.g 1, 2, 3, 4, ..

Bedrooms (int) - The number of bedrooms in the house e.g 1, 2, 3, 4, ..

Bathrooms (int) - The total number of bathroom in each house e.g 1, 2, 3, 4, ..

State (string) - The name of state of each house or the sate where the house is located. i.e Abuja, Lagos, Ogun ..

Price (float) - The total amount or price of each house e.g 2.7, 1.0. 2000.00, (million) ..

Toilet (int) - The total number of toilet in each house e.g 1, 2, 3, 4, ...

Town (string) - The name of the town where the house are built. i.e Lekki, Wuye, Jahi, ..

NB: Town will be our target variable in this instance.

Town Distribution:

  • The cleaned data still shows an imbalance in the number of listings across towns, which may affect our model's performance.

Numerical Features:

  • The descriptive statistics confirm that price values are in millions, ranging from 0.09 million to 322 million. Other features like bedrooms, bathrooms and toilets have consistent ranges without extreme outliers.

def predict_town(bedrooms, bathrooms, toilets, parking_space, title, state, price_millions):
    """
    Predicts the town based on the input features using the trained logistic regression model.
    Handles exceptions and filters predictions to only include towns within the specified state.
    """
    try:
        # Create a DataFrame with the input features
        input_data = pd.DataFrame({
            'bedrooms': [bedrooms],
            'bathrooms': [bathrooms],
            'toilets': [toilets],
            'parking_space': [parking_space],
            'title': [title],
            'state': [state],
            'price_millions': [price_millions]
        })

        # One-hot encode the categorical features
        input_data = pd.get_dummies(input_data, columns=['title', 'state'], drop_first=True)

        # Align columns with training data
        input_data = input_data.reindex(columns=X_train.columns, fill_value=0)

        # Scale the input features
        input_data_scaled = scaler.transform(input_data)

        # Make the prediction
        predicted_town_encoded = rf_classifier.predict(input_data_scaled)[0]

        # Get all possible towns for the given state from the training data
        state_towns = trimmed_data[trimmed_data['state'] == state]['town'].unique()

        if len(state_towns) == 0 :
          return "No towns found for the given state in the training data."

        # Inverse transform the encoded prediction to get the town name
        predicted_town = label_encoder.inverse_transform([predicted_town_encoded])[0]

        # Check if predicted town belongs to the specified state
        if predicted_town in state_towns:
            return predicted_town
        else:
            # If the predicted town is not in the specified state, find the closest match
            state_encoded_towns = label_encoder.transform(state_towns)
            differences = np.abs(state_encoded_towns - predicted_town_encoded)
            closest_index = np.argmin(differences)
            return state_towns[closest_index]

    except KeyError as e:
        return f"Error: Invalid input feature or state. Check your input. {e}"
    except ValueError as e:
        return f"Error: Invalid input value or data mismatch. {e}"
    except Exception as e:
        return f"An unexpected error occurred: {e}"
